2025-2-28梦熊CSPS模拟赛赛后总结

发表于2025-03-09|更新于2025-03-15

T1.put

就是有一排杆子，放倒会占用一些位置，然后问最多放倒多少

然后我就直接 dpdpdp ,正确性显然

然后他们说可以用贪心写，可以放左边就放左边

都行

T2.match

有一个二分图，问每一次删去一个点，然后求构成完美匹配的最小代价

每对点建图的代价为 max⁡{bj−ai}\max \{ b_j-a_i \}max{bj​−ai​}

然后发现其实删去每一个点后，每一个点所连的点就是，排完序后于之相对的点

求下前缀和就可以

T3.div

求 1→n1\to n1→n,每一个数的因子的和

性质题，难评，在加上一个数论分块就好

T4.path

就是不可以经过一些限定连续的边，考虑化边为点，跑最短路

但发现菊花图就炸了

但是边权都为 111 那么就可以直接 bfsbfsbfs 边算边删边控制空间

但其实我感觉没有啥影响

T5.oddeven

给定一个字符串如 s1…sns_1 \dots s_ns1​…sn​，每一次选择目前字符串的一段如 L,RL,RL,R 取出

将取出的字符串下标为偶数的放在前面 sL…sR→sL+1sL+3…sLsL+2…s_L \dots s_R \to s_{L+1} s_{L+3} \dots s_{L} s_{L+2} \dotssL​…sR​→sL+1​sL+3​…sL​sL+2​…

再将字符串插入到 RRR 之后

最后求处理完后一段区间的字符

可以考虑每一个字符是如何转移而来的，就可以很好地处理这一类构造问题

不难发现 [ri+1,ri+(ri−li+1)][r_i+1,r_i+(r_i-l_i+1)][ri​+1,ri​+(ri​−li​+1)] 是由 [li,ri][l_i,r_i][li​,ri​] 这一段复制而来，那么用 BITBITBIT 去倒着模拟这一过程就可以顺着找到每一个字符的源头
