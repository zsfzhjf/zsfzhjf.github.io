B - Ticket Gate Log

因为插入不会改变串的相对顺序

C - Variety Split Easy

分成两个前后缀数组，然后预处理，统计时合并就好

D - Cubes

给定 NNN 找 (a,b)(a,b)(a,b) 使得 a3+b3=Na^3+b^3=Na3+b3=N

因式分解下 (a+b)(a2−ab+b2)=N(a+b)(a^2-ab+b^2)=N(a+b)(a2−ab+b2)=N

然后去枚举 a+ba+ba+b 再判断是不是大概 Θ(N3)\Theta(\sqrt[3] N)Θ(3N​)

E - Path Decomposition of a Tree

看完发现就是比较典的树形 dpdpdp

看下边往不往上放就行

F - Variety Split Hard

在学校只证了满足四边形不等式，还没写

现在写完了

不等式证明

---- a ---- b ---- c ---- d ----

现需证明 w(a,c)+w(b,d)≥w(a,d)+w(b,c)w(a,c) + w(b,d) \ge w(a,d) + w(b,c)w(a,c)+w(b,d)≥w(a,d)+w(b,c)

记被分成 A,B,C,D,EA,B,C,D,EA,B,C,D,E 四个部分

左式为 ABC+CDEABC+CDEABC+CDE,右式为 ABCDE+CABCDE+CABCDE+C

若无重复数字，则显然相等

反之，则去考虑是与哪一边相重合，然后就可以贪心出来了

之后 dpdpdp 就比较显然设 dpi=max⁡{w(j,i)},j<idp_i = \max \{ w(j,i) \},j < idpi​=max{w(j,i)},j<i

对于 w(j,i)w(j,i)w(j,i) 的快速计算，因为 dpdpdp 过程的有序性，可以用类似于莫队的方法去维护 O(α(n))O(\alpha(n))O(α(n)),总复杂度就是 O(n⋅log⁡n⋅α(n))≈O(nlog⁡n)O(n \cdot \log n \cdot \alpha(n)) \approx O(n\log n)O(n⋅logn⋅α(n))≈O(nlogn)
